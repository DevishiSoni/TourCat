TourCat
- TourCat is a landmark and tourist attraction catalogue that features worldwide places and offers favouriting, searching, viewing details and seeing locations on an interactive map! You may also add your own landmark that you would like to see on the page as well as edit any existing landmarks!

Notes:
- A JSON file is used as our database to demonstrate the creation of the REST API from scratch without using a JSON server and its methods.
Express.js was used to demonstrate the manual implementation of routing setup and for its simple integration with our backend and Node server.
- No login logout was implemented so that all users can add, edit, delete and show their favourite landmarks to each other! Have fun!
- Our project and code is designed to display an efficient and organized architecture, and code is written and commented on according to methods and project format to increase maintainability, readability and security.
- The user interface is created with usability in mind with large icons, graphs and displays, and more!
- The Home Page graphical visual displays the recorded countries represented. Landmarks can be added with limited information, so please add as much as you can!

Features:
- Main home page with statistics regarding the landmarks
- Landmarks page with a view of all landmarks, an option list for each landmark (view details, edit, delete, favourite) and an interactive map
- Searching and filtering (filter by location, type, country)
- Favourites list
- Landmarks adding page with image option
- Specific landmark pages to view unique details and images Contact us page

To run our webpage:
- Run the server
  - cd into the server folder
  - npm install
  - node server.js

- Run the client
    - As a production webpage
      - cd into the client folder
      - npm install
      - npm run build
      - npm run preview
      - Open the link provided in the terminal.
    - As a development webpage
      - cd into the client folder
      - npm install
      - npm run dev
      - Open the link provided in the terminal.

Enjoy TourCat!
