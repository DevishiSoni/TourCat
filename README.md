# TourCat

# To run Front End
- Cd into client folder
- run 'npm install'
- run npm run dev
- if running using npm build, adjust file routing in server.js, other file path conflicts may arise

# To run backend (server):
- Cd into server folder
- Run 'npm install'
- Run 'node server.js'
- Open 'http://localhost:3000/api/test'
